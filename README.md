# Drawing Prompt Master (绘画提示词大师) 🎨

这是一个基于 Manus Skill 规范构建的绘画提示词专家库。它不仅提供高质量的提示词模板，更核心的功能是通过**多轮交互**引导用户进行创意“冷启动”，解决“不知道画什么”或“不知道怎么描述”的痛点。

## 🌟 核心特性

- **多轮交互引导**: 支持从模糊意向到具体指令的渐进式构思。
- **热门 IP 库**: 预设海贼王、咒术回战、火影忍者等热门动漫 IP 的核心特征描述。
- **风格多样化**: 涵盖日系动漫、电影感群像、赛博朋克、经典油画等多种主流视觉风格。
- **排版与构图**: 专门优化了带有 Typography（排版文字）和高角度俯视等高级构图的提示词生成。

## 📂 仓库结构

```text
drawing-prompt-master/
├── SKILL.md              # Skill 核心指令，包含多轮交互逻辑
├── references/
│   ├── ip_characters.md  # 热门 IP 角色特征参考库
│   └── style_library.md  # 绘画风格与视觉效果库
├── scripts/              # 自动化脚本工具
└── templates/            # 提示词模板资源
```

## 🛠 如何使用

### 在 Manus 中使用
1. 将本仓库内容加载为 Manus Skill。
2. 告诉 Manus 你的模糊想法（例如：“我想画火影忍者”）。
3. 按照 Manus 的引导，逐步确定角色、场景和风格。
4. 获取最终生成的 GPT Image 或 Nano Banana 提示词。

### 交互示例
> **用户**: 我想画个火影忍者的图，但不知道画什么。
>
> **Manus**: 没问题！我们可以一起构思一个震撼的画面。你更倾向于哪种方向？
> 1. **经典对决**: 比如鸣人与佐助在终结之谷的最后激战。
> 2. **现代潮流**: 角色穿上现代潮流服饰，站在带有大排版文字的极简广场上。
> 3. **温馨日常**: 木叶村日常，比如一乐拉面店里的聚会。
> 
> 你有特别喜欢的角色或者想要尝试的画风吗？
